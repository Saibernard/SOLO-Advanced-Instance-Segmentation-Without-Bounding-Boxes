# SOLO-Advanced-Instance-Segmentation-Without-Bounding-Boxes
Implemented the SOLO (Segmenting Objects by Locations) framework for efficient instance segmentation across 3265 images featuring vehicles, people, and animals. The model, enhanced with a Feature Pyramid Network, directly predicts segmentation masks, eliminating the need for bounding box proposals and achieving precise object localization and classification.
